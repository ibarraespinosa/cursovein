Começando

Obrigado por começar este curso. Para instalar vein você precisa instalar primeiro R desde https://cran.r-project.org/. R pode ser instalado em diferentes sistemas operacionais.

IDEs como RStudio o VSCode

É altamente recomendado instalar Rstudio ou VSCode.

Windows

Instalar Rtools, que provê MinGW-w64 32/64-bit para compilar C, Fortran e C++. Isto é importante porque o vein tambem possui alguns códigos em Fortran.
