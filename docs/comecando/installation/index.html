Instalação

Instalação de VEIN

A forma mas facil de instalar vein é, abrir R e escrever na termina:

Com a versão de GitHub. O curso vai usar a versão 0.8.8

#install.packages("remotes")
remotes::install_github("atmoschem/vein")

ou a versão do CRAN

install.packages("vein")

Outras librerias

Pode ser que os usuarios de Windows tem que primeiro instalar raster de CRAN

install.packages("raster")
remotes::install_github("atmoschem/eixport")

E as outras librerias assim:

install.packages(c("ggplot", "readxl"))

Dica

Antes de usar VEIN, fecha Google Chrome pois usa muita memoria RAM.
