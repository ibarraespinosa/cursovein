Pre-Requisitos

vein importa funções de:

  • sf, para trabalhar com vetores espaciais.
  • data.table processador de data.frame ultra rápido.

Outras librerias necessarias para o curso

  • eixport: lê as saídas espaciais do VEIN e outros para gerar dados de entrada de modelos de qualidade do ar, como WRF-Chem.
  • readxl: lê arquivos de Excel.
  • ggplot2: faz gráficos de qualidade de publicação.

Spatial Features sf

sf é um pacote de R com interface para as bibliotecas GDAL (vetores), PROJ e GEOS. Isto implica que estas bibliotecas precisam estar presentes no sistema antes de instalar sf. eixport importa funções do ncdf4, portanto, também é necessário instalar NetCDF.

As recomendações da página web de GitHub sf para instalar as bibliotecas em diferentes sistemas são:

Windows

Instala [R], ve a versão e instala Rtools, que provê MinGW-w64 32/64-bit para compilar C, Fortran e C++. Isto é importante porque o VEIN também possui códigos em Fortran.

Ubuntu

sudo apt-get install libudunits2-dev libgdal-dev libgeos-dev libproj-dev libnetcdf-dev

Fedora

sudo dnf install gdal-devel proj-devel proj-epsg proj-nad geos-devel udunits2-devel netcdf-devel

Arch

pacman -S gdal proj geos netcdf
pacaur/yaourt/whatever -S udunits

MacOS

brew install pkg-config
brew install gdal netcdf
